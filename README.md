# Project REST-Rant

REST-Rant is an app where users can review restaurants.

ROUTES

| Method    | Path                     | Purpose           |
|-----------|--------------------------|-------------------|
| GET       | /                        |                   |
| GET       | /places                  |                   |
| POST      | /places                  |                   |
| GET       | /places/new              |                   |
| GET       | /places/:id              |                   |
| PUT       | /places/:id              |                   |
| GET       | /places/:id/edit         |                   |
| DELETE    | /places/:id              |                   |
| POST      | /places/:id/rant         |                   |
| DELETE    | /places/:id/rant/:rantId |                   |
| GET       | *                        |                   |
